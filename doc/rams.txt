RAMS, a minimalistic colorscheme.

==============================================================================
Color scheme options					rams-options*

						*g:rams_transp_bg*
Set to 1 if you want a transparent background. Takes effect only in the
terminal.
>
	let g:rams_transp_bg = 0
<
